# Personal Portfolio Website

A modern, enterprise-style portfolio dashboard built with React, TypeScript, and Express.

## Features

- **Dashboard-style UI**: Professional, clean, and data-driven design.
- **Projects Showcase**: Detailed project cards with live demo and GitHub links.
- **Skills Management**: Grouped technical skills categorization.
- **Certifications**: Professional credentials with verification links.
- **Contact System**: Full-stack contact form with email notifications via Resend.
- **Theme Support**: Dark and light mode toggle.

## Tech Stack

- **Frontend**: React, TypeScript, Vite, Wouter, Tailwind CSS, shadcn/ui.
- **Backend**: Express.js, Drizzle ORM, PostgreSQL.
- **Integrations**: Resend for email notifications.

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set up your environment variables (DATABASE_URL, RESEND_API_KEY).
4. Push the database schema: `npm run db:push`.
5. Start the development server: `npm run dev`.
